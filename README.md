# busylight-cli

Set your kuando busylight - no messing

```
npm install busylight-cli -g
```

Set your light red
```
busylight busy
```

Set your light green
```
busylight free
```

Set your light to your mood
```
busylight pink
```

## effects

Pulse your light
```
busylight busy --pulse
```

Or blink instead
```
busylight busy --blink
```
